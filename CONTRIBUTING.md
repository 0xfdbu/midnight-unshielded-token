# Contributing to midnight-unshielded-token

Thank you for your interest in contributing to this project. We welcome contributions from developers of all skill levels.

## How to contribute

1. **Fork the repository** and create a feature branch.
2. **Make your changes** following the existing code style.
3. **Open a pull request** with a clear description of what you changed and why.

## What we accept

- Improvements to the tutorials, documentation, or code
- Bug fixes and performance optimizations
- New test scripts or tooling
- Corrections that keep code snippets accurate and fact-checked against the actual source files

## Content restrictions

We do not accept contributions related to any of the following:

- **Gambling** or games of chance
- **Lending**, borrowing, or leveraged trading platforms
- **Prediction markets** or betting mechanisms
- **Inappropriate or harmful content**, including but not limited to:
  - Hate speech, discrimination, or harassment
  - Violence, threats, or illegal activities
  - Adult or explicit material
  - Scams, fraud, or deceptive practices

Pull requests that include any of the above will be closed without review.

## Code style

- Use TypeScript for all new code.
- Follow the existing project structure.
- Keep code snippets in tutorials accurate and fact-checked against the actual source files.
- Use sentence case for headings and active voice for explanations.

## Questions?

Open an issue or reach out in the Midnight developer community.
